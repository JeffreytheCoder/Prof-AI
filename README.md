# Prof AI

Turn any textbook into your personal AI professor. View more details on [Devpost](https://Devpost.com/software/prof-ai). 

## Run 

```
cd backend  
pip install -r requirements.txt
uvicorn --app-dir=./app main:app --reload
```

```
cd frontend
npm i
npm start
```

```
cd frontend-qa
npm i
npm run dev
```
